Como usar
  1. Cadastre o cliente com ID da conta de anúncio (ex: act_123456789), Access Token (ads_management) e ID da Página
  2. Use "Criar Completo" no menu Campanhas — um wizard de 3 passos: Campanha → Segmentação → Criativo
  3. No Passo 1: nome, objetivo (Vendas/Leads/Tráfego) e orçamento diário
  4. No Passo 2: cidades, idade, gênero, interesses do público
  5. No Passo 3: imagem, headline, texto, link e botão CTA
  6. Revise e crie — tudo sai pausado para revisão antes de ativar
